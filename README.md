<!-- Building a Network Command Line Interface Tool In Go
 -->


```
go run cli.go ip --host google.com

NAME:
   Website Lookup CLI - Let's you query IPs, CNAMEs, MX records and name servers

USAGE:
   cli [global options] command [command options] [arguments...]

COMMANDS:
   ns       Looks up the Name servers for Particular host
   ip       Looks up the IP Address for Particular host
   cname    Looks up the CNAME for Particular host
   mx       Looks up the MX for Particular host
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help
```
